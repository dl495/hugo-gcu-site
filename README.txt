* * * HOW TO UPDATE REPOSITORY * * *


1. Navigate to /Site

2. git add *
   - this adds all the files/folders not in /public to the list

3. git push origin master
   - this pushes the content to the repository

4. git commit -a -m "<message>"
   - this commits and records the changes made


* * * * * *

Update site itself with the Website_updater.command batch file :)
